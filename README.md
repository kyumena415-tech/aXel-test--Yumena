# 使ったコマンドとメモ

- git checkout -b feature-branch : 新しいブランチを作成
- git add . : 変更したファイルを登録
- git commit -m "コメント" : 変更を記録
- git push origin feature-branch : ブランチをGitHubにアップロード
git add README.md
git commit -m "ファイル名をREADME.mdに変更"
git push origin main
